A fun test to determine if you're Gen X or Millennial based on your Winona Ryder associations.

== Description ==

This plugin provides a shortcode [winona_ryder_test] that displays a quiz asking users what comes to mind when they think of Winona Ryder, then tells them whether their answer suggests they're Gen X or Millennial.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/winona-ryder-test` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the [winona_ryder_test] shortcode on any page or post

== Frequently Asked Questions ==

= What answers count as Gen X vs Millennial? =

The answers are based on Winona Ryder's filmography and public persona during different eras.

= Can I customize the answers? =

Yes, you can edit the Millennial-Answers.txt and GenX-Answers.txt files in the plugin directory.
